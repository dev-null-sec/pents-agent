## 速率限制与安全

- DNS 并发/速率：R001 前置验证参数见 E-0009；R003 正式执行使用 massdns direct，167377 候选、60.669 秒，resolver 见 E-0011
- HTTP/CDN/服务指纹速率：由 Claude Code 执行时自定安全区间；建议全局并发 1-2，每个请求之间随机等待 2-8 秒，每个子域名首轮请求预算不超过 8 次，遇到 429、WAF 拦截、连续 5xx、明显业务异常立即暂停
- 小范围 Web 端口确认速率：仅测 80/443/8080/8443 等常见 Web 入口端口；全局并发 1，目标之间随机等待 3-10 秒；不做全端口扫描和服务爆破
- 路径/API 验证速率：仅在已存活入口上验证少量高价值候选，例如 `/setup`、`/api/v1/setup/status`；请求之间随机等待 5-15 秒，不使用目录/API 大字典
- 高风险操作需确认：是
- 数据处理说明：只记录最小必要证据；不得保存真实敏感数据、凭据或个人信息
